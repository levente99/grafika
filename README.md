### Használat

- Mozgatás
	>W A S D billentyűk
- Kamera forgatás
	>Jobb vagy bal klikk + egér mozgatás
- Kamera magasság növelése
	>SPACE
- Kamera magasság csökkentése
	>C
- Fényerő növelés/csökkentés
	> " + " , " - " billentyűk
- Szék mozgatása
	>X
- Szék forgatása jobbra
	>E
- Szék forgatása balra
	>Q
- HELP menü
	>F1 billentyű
- Kilépés
	>Esc billentyű
